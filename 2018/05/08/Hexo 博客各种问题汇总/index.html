Hexo 博客各种问题汇总

发表于 2018-05-08 | 更新于 2018-07-29 | 分类于 经验

在整理这几天与 Hexo 博客相关的操作时，发现有一些细节是需要多加注意的，可能它们很细微，但是一旦没有处理好，其后果将会非常麻烦，所以专门写一篇文章将这些小点汇集在一起，便于以后查阅。

一些概念

  • -config.yml代表与配置相关的文件，在博客文件夹中有两处：

    1. 全局配置文件：在你建的博客文件夹下，这是确定整个博客站点设置的文件，比如语言、主题和站名。我的是 D:\blog\lerinaz.github.io\_config.yml
    2. 主题配置文件：在下载-解压的主题文件夹中，确定所用主题的相关设置，主要用于更改页面设置，比如增加标签、分类或者其他页面。我的是 D:\blog\lerinaz.github.io\themes\next\_config.yml

更换语言

  • 要注意主题配置文件中的文件夹languages，根据其中的设定来设置所用语言的代码。

    我在修改语言时首先根据官方文档将汉语配置为zh-Hans

    然而当我打开博客——

    ？？这是什么语言！然后到处查阅，却都是和官方一样的做法，四处乱碰了十来分钟后终于在主题文件夹下找到一个名为languages的文件夹，以超级紧张的心情打开它——哇有戏！凭着我多年来和语言打交道的经验，一个名为zh-CN.yml的文件吸引了我的注意力，打开果然是汉语。于是返回去在全局配置文件中将language:修改为zh-CN——啊哈成功啦！

    然后赶紧找师兄报告这个消息，结果他告诉我他之前做的时候凭感觉配了zh-CN，直接炸掉，后来根据官方文档改成zh-Hans才好了的。……我不是很明白这中间到底是怎么一回事，但是不管怎样，按照主题设置中的语言文件夹来肯定没错，所以最保险的做法是先查看所需语言的代码，再进行修改。

添加标签页和分类页

  • 首先要在博客文件夹下运行命令hexo new page tags以开启标签页，否则之后标签页会打不开
  • 步骤：开启标签页；修改主题配置文件；修改标签文件夹的设置；在文章中增加tag

关于字号

  • NexT 主题将标签字号大小与标签数量相关，相关文章越多的标签字号会相对大一些。
  • 有时候文章排版格式显得有点奇怪，可能是浏览器设置的网页缩放的原因，在修改之前不如先确认一下网页缩放的设置。比如我就被 Edge 默认的125%缩放值坑了一把。

相关链接

关于 Hexo 博客的搭建和管理，网上有非常多的教程，我所记录的只是其中极其基础和简单的操作。因为很赞同师兄的一句话：“博客最重要的是内容”，所以我对自己的博客设置所做的改动并不很多。

下面是一些我用到过的教程，它们覆盖的操作面非常广，如果你希望自己的博客更加个性化，可以作为参考：

  • 博客“眼前的苟且”：https://javaclear.github.io/ 这位博主写了 Hexo 博客使用的一系列文章，并且写得很详细，我这几天几乎是跟着他一样一样走下来的。

  • 博客“Lee’s Blog”https://mrlrf.github.io/2017/05/05/Hexo-github搭建博客/ 其实，我好喜欢那个飘散的丝丝缕缕的动画的……点击一处慢慢移动还能将更多的细丝汇聚起来，等变成一个超大的线球时再一下子脱开。周末我就这样玩了好一会儿……

  • NexT 主题官方文档 http://theme-next.iissnan.com/ 我用的是 NexT 主题，中途遇见了一些问题，师兄直接把官方文档甩给了我，所以我也不知道他从哪里寻觅来的。

在博客的搭建和管理过程中可能会遇见各种各样的问题，这时候如果你 Google 一下的话会有惊喜哦！基本上所有的问题都能在网上找到解决方法。

提醒自己

  • 注意命令格式：hexo g && hexo d
  • 表示文件夹的层级时，只有 Windows 系统是“\”
  • 之后涉及到修改域名和图标或者其他功能时，CNAME 和 favicon.ico这样的文件要放在source文件夹下，否则重新上传时会被删掉。

最后

就我的经历来看，搭建和经营自己的博客是一件很奇妙的事，它让我知道原来有那么多那么多有趣的东西，也让我尝试着去做一些从前连想都没有想过的事情。也祝愿每一个和我一样，经营着自己的小小世界的人享受这份快乐~~

P.S. 据师兄自己所说，他这个带我上路、全程指导我的人竟然都好久不更新自己的博客了……
